Este repositorio documenta el desarrollo de un ejercicio de analítica predictiva orientado a la detección de fraude en transacciones con tarjeta de crédito utilizando el dataset creditcard.csv. El objetivo principal es replicar un escenario real de análisis con información limitada sobre las variables disponibles, priorizando la formulación de preguntas relevantes, la exploración de patrones y la interpretación de resultados desde una perspectiva operativa.

El conjunto de datos contiene transacciones anonimizadas mediante transformación PCA, donde las variables explicativas (V1–V28) representan componentes latentes y las únicas variables originales disponibles son el importe y el momento de la operación. La variable objetivo identifica si la transacción es fraudulenta o legítima, permitiendo desarrollar modelos de clasificación aun sin conocer el significado explícito de los predictores.

El trabajo incluye exploración numérica y visual para analizar la relación entre variables y fraude, identificación de patrones relevantes y construcción de modelos predictivos siguiendo metodologías estándar de ciencia de datos. Se presta especial atención a la interpretabilidad del resultado desde una perspectiva de negocio, trasladando las conclusiones técnicas a implicaciones operativas comprensibles para perfiles no técnicos, tales como la caracterización de comportamientos de riesgo o la identificación de segmentos con mayor probabilidad de fraude.

El análisis se desarrolla en entorno R utilizando herramientas de manipulación y visualización de datos que priorizan claridad, reproducibilidad y comunicación efectiva de resultados, configurando el repositorio como un ejercicio de integración entre modelización técnica y generación de valor analítico para la toma de decisiones.
